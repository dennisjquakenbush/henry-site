# Henry Quakenbush — Personal Site

A unique, single-page personal résumé site for **Henry Quakenbush** — operator, builder, and marketer.

- **Live site:** _(GitHub Pages URL appears here once Pages is enabled)_
- **Featured project:** [Libera News](https://liberanews.com)

## Features

- Kinetic serif hero with animated intro
- Interactive spotlight cursor + ambient orbs
- Scroll-reveal animations & animated stat counters
- Ventures grid, experience timeline, filterable skills matrix
- Featured Libera News module (Four Freedoms)
- Fully responsive, no build step, zero dependencies (single `index.html`)

## Deploy (GitHub Pages)

1. Push this folder to a GitHub repository.
2. Repo **Settings → Pages** → *Build and deployment* → **Deploy from a branch**.
3. Select branch `main` and folder `/ (root)` → **Save**.
4. For a personal site at `<username>.github.io`, name the repository `<username>.github.io`.

## Local preview

```bash
python3 -m http.server 4599

```
